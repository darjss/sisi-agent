\chapter*{УДИРТГАЛ}
\addcontentsline{toc}{chapter}{УДИРТГАЛ}

Орчин үеийн хэлний загварууд (Large Language Models) текст боловсруулах, асуултад хариулах, код бичих зэрэг олон төрлийн даалгаврыг гүйцэтгэх чадвартай болсон. Гэсэн хэдий ч эдгээр загварууд ``хаалттай орчин''-д ажилладаг тул бодит системүүдтэй шууд харилцах, амьд өгөгдөлд хандах, хэрэглэгчийн өмнөөс үйлдэл гүйцэтгэх боломжгүй. Өөрөөр хэлбэл, загварын суурь оюун ухаан хангалттай боловч гаднах ертөнцтэй харилцах ``гар, хөл'' дутагдалтай байна.

Энэхүү бакалаврын судалгааны ажил нь тус асуудлыг шийдвэрлэхийн тулд Монгол Улсын Их Сургуулийн SISI оюутны порталд зориулсан хиймэл оюун бүхий туслах систем (SISi Intelligent Agent) бүтээхэд чиглэж байна. Уг систем нь Model Context Protocol (MCP) стандартыг ашиглан хэлний загварт порталын бодит өгөгдөлд хандах, шаардлагатай үйлдлийг гүйцэтгэх хэрэгслүүдийг олгоно.

Завсрын тайлангийн хүрээнд дараах ажлуудыг гүйцэтгэсэн:

\begin{itemize}
  \item SISI порталын API-д холбогдох MCP серверийг бүрэн хэрэгжүүлж, найман read-only хэрэгсэл (tool) тодорхойлсон
  \item Chrome extension-ийн хэрэглэгчийн интерфейсийг боловсруулсан
  \item AI серверийн суурь бүтцийг бий болгож, хэлний загвартай холбосон
  \item Системийн ерөнхий архитектур, модулийн бүтэц, session удирдлагын механизмыг загварчилсан
\end{itemize}

Дараагийн шатанд AI сервер болон MCP серверийг бүрэн холбож, хэрэглэгчийн байгалийн хэлний хүсэлтийг бодит системийн үйлдэл болгон хувиргах tool orchestration-ийг хэрэгжүүлэх, Chrome extension-оос session bootstrap хийх урсгалыг дуусгах, мөн багшийн порталын API интеграцыг нэмэх зэрэг ажлууд үлдэж байна.

Энэхүү тайлан нь төслийн зорилго, судалгааны үндэслэл, онолын суурь, системийн шаардлага, архитектурын загварчлал, технологийн сонголт, одоогийн хэрэгжилтийн төлөв, болон цаашдын төлөвлөгөөг тус тус хамарна.

\chapter*{ДҮГНЭЛТ}
\addcontentsline{toc}{chapter}{ДҮГНЭЛТ}

Энэхүү завсрын тайлангаар SISi Intelligent Agent төслийн судалгааны үндэслэл, онолын суурь, техникийн архитектур, болон одоогийн хэрэгжилтийн төлөвийг танилцуулав.

Төслийн гол санаа нь орчин үеийн хэлний загварууд аль хэдийн хангалттай ``оюун ухаан''-тай болсон боловч бодит системүүдтэй харилцах хэрэгслүүд дутагдалтай байдгийг шийдвэрлэхэд оршино. Загварын чадамжийг нэмэгдүүлэхийн оронд түүнд зохих хэрэгслүүдийг олгох нь илүү үр дүнтэй гэсэн таамаглалыг энэхүү төсөл баталгаажуулахыг зорьж байна.

Завсрын шатанд гарсан гол үр дүнгүүд:

\begin{enumerate}
  \item \textbf{MCP сервер бүрэн ажиллагаатай:} SISI порталын API-тай холбогдох локал MCP сервер найман read-only хэрэгсэлтэй ажиллаж байна. Оюутны мэдээлэл, хуваарь, дүн, санхүү, мэдэгдэл зэрэг үндсэн домэйнүүдийг хамарсан.
  
  \item \textbf{Session удирдлага хэрэгжсэн:} Хэрэглэгчийн SISI сессийг баталгаажуулах, хадгалах, шинэчлэх механизм бүрдсэн. Энэ нь дахин нэвтрэлт шаардахгүй хэрэглээний суурь болно.
  
  \item \textbf{Chrome extension интерфейс:} Хэрэглэгчийн чат интерфейс бүхий extension-ийн UI бэлэн болсон. Одоогоор mock хариултуудтай ажиллаж байгаа боловч бүтэц нь бэлэн.
  
  \item \textbf{AI серверийн суурь:} Elysia framework дээр AI endpoint ажиллаж, Gemini загвартай холбогдсон. Streaming хариулт авах боломжтой.
  
  \item \textbf{Модульчлагдсан архитектур:} Monorepo бүтцээр apps болон packages-ийг тусгаарлаж, цаашдын өргөтгөлд бэлэн суурь тавигдсан.
\end{enumerate}

Цаашдын ажлын гол чиглэлүүд:

\begin{itemize}
  \item AI сервер дээр MCP хэрэгслүүдийг дуудах tool orchestration-ийг хэрэгжүүлэх
  \item Chrome extension-оос SISI порталын session-ийг автоматаар авч MCP серверт дамжуулах
  \item Багшийн порталын API-д зориулсан MCP хэрэгслүүд нэмэх (Excel-ээс дүн оруулах)
  \item Mutation үйлдлүүдийн баталгаажуулалт, audit trail хэрэгжүүлэх
  \item Хэрэглэгчийн туршилт, үнэлгээ явуулах
\end{itemize}

Энэхүү завсрын тайлан нь техникийн суурь амжилттай бүрдсэнийг харуулж байгаа бөгөөд эцсийн шатанд бүрэн ажиллагаатай AI туслах системийг хэрэглэгчдэд хүргэх боломжтой гэдгийг нотолж байна.
