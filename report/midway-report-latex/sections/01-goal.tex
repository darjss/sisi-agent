\chapter{СУДАЛГААНЫ ЗОРИЛГО}

Энэхүү бакалаврын судалгааны ажлын зорилго нь Монгол Улсын Их Сургуулийн SISI оюутны порталын өгөгдөл болон үйлчилгээг хиймэл оюун бүхий байгалийн хэлний интерфейсээр дамжуулан хүргэх туслах систем (SISi Intelligent Agent) загварчлах, хөгжүүлэх, турших явдал юм.

Төслийн үндсэн зорилго нь хэлний загварын (LLM) суурь чадамжийг ашиглан оюутны өдөр тутмын даалгаврыг автоматжуулах, мэдээллийн тархай байдлыг нэгтгэх, хэрэглэгч болон порталын хоорондын харилцааг хялбарчлах явдал юм. Энэ нь зөвхөн асуултад хариулах чатбот бус, бодит системтэй холбогдож үйлдэл гүйцэтгэх чадвартай ``агент'' системийг бий болгохыг зорьж байна.

\section{Зорилтууд}

Дээрх зорилгод хүрэхийн тулд судалгааны болон техникийн хэд хэдэн зорилтыг дэвшүүлж байна.

\subsection{Судалгааны зорилтууд}

Нэгдүгээрт, SISI порталыг ашиглах үед оюутнуудын тулгардаг бэрхшээл, давтагддаг үйлдлүүд, цаг хугацааны алдагдлыг тодорхойлох шаардлагатай. Хоёрдугаарт, AI agent архитектур, tool-calling механизм, Model Context Protocol (MCP) стандартын онолын үндсийг судлах хэрэгтэй. Гуравдугаарт, боловсролын салбарын AI туслах системүүд, tool-use системүүд, байгууллагын автоматжуулалтын шийдлүүдийг харьцуулан үнэлэх нь чухал.

\subsection{Техникийн зорилтууд}

Техникийн хувьд эхлээд SISI порталын API-д холбогдох Model Context Protocol нийцтэй сервер хөгжүүлэх шаардлагатай. Үүний зэрэгцээ хэрэглэгчийн байгалийн хэлний оролтыг хүлээн авах, хариултыг харуулах Chrome extension бүтээх, мөн хэлний загвар болон MCP хэрэгслүүдийг холбох orchestration давхарга хэрэгжүүлэх хэрэгтэй. Хэрэглэгчийг дахин нэвтрүүлэхгүй, одоогийн SISI сессийг аюулгүй дамжуулах механизм бий болгох нь мөн чухал зорилт юм. Эцэст нь багшийн зориулалттай API интеграц, тухайлбал Excel-ээс дүн оруулах боломжийг нэмэх төлөвлөгөөтэй.

\subsection{Үнэлгээний зорилтууд}

Үнэлгээний хувьд системийн үндсэн use case-үүдийг туршиж амжилтын хувийг хэмжих, хэрэглэгчийн туршилтаар ашиглах чадвар болон сэтгэл ханамжийг үнэлэх, мөн аюулгүй байдлын шаардлагуудыг хангаж буй эсэхийг шалгах зорилтуудыг тавьж байна.

\section{Хүрэх үр дүн}

Судалгааны ажлын эцэст хэд хэдэн үр дүнд хүрэхийг зорьж байна. Үүнд SISI порталын өгөгдлийг байгалийн хэлээр асууж авах боломжтой ажиллагаатай прототип, хуваарь, дүн, санхүү, мэдэгдэл зэрэг домэйнүүдийн read-only MCP хэрэгслүүд, баталгаажуулалттай бичих үйлдлүүд болох CSV export болон дүн оруулах mutation хэрэгслүүд, бодит оюутнуудын санал хүсэлт дээр үндэслэсэн хэрэглэгчийн туршилтын тайлан, мөн архитектур, API, хэрэгжүүлэлтийн техникийн баримт бичиг багтана.

\section{Хамрах хүрээ}

Энэхүү бакалаврын ажлын хүрээнд SISI оюутны порталын үндсэн өгөгдөл буюу хуваарь, дүн, санхүү, мэдэгдэл, хувийн мэдээллийг хамруулна. Мөн оюутны талаас хийх read-only асуулга, баталгаажуулалттай mutation үйлдлүүд, багшийн порталын дүн оруулах үйлдэл, Chrome extension-д суурилсан хэрэглэгчийн интерфейс зэргийг багтаана.

Харин бүх NUM системүүдийн бүрэн интеграц, mobile application хөгжүүлэлт, олон хэрэглэгчийн нэгэн зэрэг ашиглах production deployment, порталын аюулгүй байдлын бүрэн аудит зэргийг энэхүү судалгааны ажлын хүрээнд хамруулахгүй болно.
