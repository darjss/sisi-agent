\chapter{АШИГЛАЛТЫН ХУВИЛБАРУУДЫН ДЭЛГЭРЭНГҮЙ ЖАГСААЛТ}

Энэ хавсралтад төслийн эхний шатанд тодорхойлогдсон хэрэглэгчийн чухал командуудыг нэгтгэн үзүүлэв. Эдгээр нь цаашдын хэрэгжилт болон туршилтын сценари боловсруулах үндэс болно.

\section{Оюутны талын үндсэн хувилбарууд}

\begin{itemize}
  \item SISI дээрх миний хичээлийн болон шалгалтын хуваарийг нэг дор гаргаж тайлбарлах.
  \item Миний даалгавар, deadline, мэдэгдэл зэрэг мэдээллийг MCP-ээр холбогдсон бусад систем рүү дамжуулах.
  \item Миний үзээгүй эсвэл дутуу үлдсэн хичээлүүдийг харуулах.
  \item Дараагийн семестрт үзэх боломжтой хичээлүүдийг санал болгох.
  \item Миний хичээлийн дүнг CSV хэлбэрээр экспортлох.
\end{itemize}

\section{Ерөнхий MCP өргөтгөлийн хувилбарууд}

\begin{itemize}
  \item MCP-ээр холбогдсон календарь, баримт бичиг, мэдэгдлийн системүүд рүү өгөгдөл дамжуулах.
  \item Бэлэн болсон мэдээллийг шаардлагатай файл эсвэл системийн хэлбэрт хөрвүүлж дараагийн үйлдэлд ашиглах.
\end{itemize}

\section{Багшийн талын өргөтгөлийн хувилбарууд}

\begin{itemize}
  \item Багшийн порталын API-тай холбогдож Excel файлаас олон оюутны дүн оруулах.
  \item Дүн оруулахын өмнө мөр бүрийн шалгалт, алдаа, баталгаажуулалтыг хийх.
  \item Оруулсан дүнгийн үр дүн, амжилтгүй мөрүүдийн тайланг гаргах.
\end{itemize}

\section{Тэмдэглэл}

Эдгээр use case-уудаас зарим нь одоогийн кодын сан дээр read-only мэдээлэл авах түвшинд боломжтой болсон бол зарим нь ерөнхий MCP өргөтгөл, бичих үйлдлийн баталгаажуулалт, багшийн порталын интеграц, Excel-д суурилсан mutation workflow зэрэг дараагийн шатны ажлуудтай холбоотой юм.
