# Midway Report

This folder contains the interim bachelor thesis report in LaTeX.

## Build

```bash
make
```

The build uses `tectonic` and writes the compiled PDF to `main.pdf`.

If the diagram PNG files need to be regenerated:

```bash
make diagrams
```

## Notes

- The report body is written in Mongolian.
- Student and advisor metadata are centralized at the top of `main.tex`.
- If you later provide the official university logo, it can be added to the title pages without changing the section content.
